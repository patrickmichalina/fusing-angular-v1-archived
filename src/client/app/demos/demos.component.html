SVGs

Server rendered SVGs are very effecient. They are also flicker free! But having a large set of SVGs stored in an angular service penalizes users by making them download a large app bundle. In order to fix this, we provide a server and client variant of the SVG service. Initial rendered SVGs come from the server and client routed SVGs are an async fetch.
